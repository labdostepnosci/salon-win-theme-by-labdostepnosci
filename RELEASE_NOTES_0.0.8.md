# Release 0.0.5

Ujednolicono warstwę wizualną podstron, wpisów i archiwów z aktualną stroną główną Salon Win.

## Changed
- Dopasowano szablony `page.php`, `single.php`, `index.php`, `404.php` i `woocommerce.php` do wspólnego systemu wizualnego.
- Dodano style dla treści długich, nagłówków, akapitów, zdjęć, galerii, cytatów, przycisków, paginacji i nawigacji wpisów.
- Dodano style dla podstawowych bloków Gutenberg oraz popularnych bloków Kadence i UAGB.
- Poprawiono responsywność widoków stron, wpisów, archiwów, galerii i bloków kolumnowych.
- Dodano widoczny fokus dla elementów interaktywnych.
- Podbito wersję motywu do `0.0.5`.

## Preserved
- Nie zmieniono treści merytorycznych, slugów, menu, danych kontaktowych ani linków zewnętrznych.
- Nie zmieniono mechanizmu rezerwacji.
- Nie zmieniono mechanizmu GitHub Releases / updatera.
- Zachowano `GitHub Theme URI` i `Primary Branch`.

## Deployment
- Przed publikacją release sprawdzić widok strony głównej, stronę „O nas”, pojedynczy wpis, archiwum wpisów, galerię, widok WooCommerce i mobile.
- Po akceptacji utworzyć tag `v0.0.5`, paczkę `salon-win-theme-0.0.5.zip` z katalogiem głównym `salon-win/` i GitHub Release.
